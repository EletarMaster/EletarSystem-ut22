([[Типы способностей#Активная способность|А]]) [[Действия#Стандартное действие|Стандартное действие]]. Персонаж или 1 союзник на дальности 10 [[Клетка|клеток]] тут же восстанавливает [[Здоровье]]. 

Эффективность молитвы зависит от результата броска [[Вера|Веры]] персонажа при ее использовании. 

К итоговому [[Восстановление Ресурсов|Восстановлению Ресурсов]] также прибавляется: [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа. 

Таблица восстановления [[Здоровье|Здоровья]] и результата броска [[Вера|Веры]]:

1. Результат 5 = 2d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа. 
2. Результат 10 = 3d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа.
3. Результат 15 = 4d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа.
4. Результат 20 = 5d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа.
5. Результат 25 = 6d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа.
6. Результат 30 = 7d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа.
7. Результат 35 = 8d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа.
8. Результат 40 = 9d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа.
9. Результат 45 = 10d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа.
10. Результат 50 = 11d6 [[Здоровье|Здоровья]] + [[Интеллект|Инт]] / [[Характер|Хар]] / [[Обаяние|Оба]] персонажа.

Эта способность работает по [[Правила Жизни и Смерти#Жизнь и Нежить|Правилам Жизни и Смерти]].

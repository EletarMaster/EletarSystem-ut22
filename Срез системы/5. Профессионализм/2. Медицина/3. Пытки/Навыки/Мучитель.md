([[Типы способностей#Пассивная способность|П]]) [[Действия#Пассивный эффект|Пассивный эффект]]. 

1. [[Медицина]]: +1. 
2. Знание: [[Список Знаний#Медицина|Медицина]]: +1.
3. Применяя [[Первая помощь#Восстановление здоровья|Восстановление здоровья]] с помощью [[Первая помощь|Первой помощи]], персонаж может наносить цели [[Урон]] по выбранным ресурсам, а не восстанавливать их. 
4. В таком случае - это считается [[Вид Воздействия#Негативный Эффект|Негативным]] - [[Тип Воздействия#Физический|Физическим]] эффектом: [[Нанесение урона]] с источником: [[Источник#Физический|Физический]]. 
5. Цель может сопротивляться пройдя [[Встречная проверка|встречную]] проверку [[Тело|Тела]], против проверки [[Медицина|Медицины]] персонажа. 
6. Этот урон является: [[Урон#Внутренний|внутренним]]. 

Изменение характеристик:
1. Теперь персонаж может выбрать [[Тело]] в качестве характеристики для бросков [[Медицина|Медицины]].

Примечание: усиление эффективности [[Первая помощь#Восстановление здоровья|Восстановления здоровья]] также усиливает эффективность нанесения Урона. Например способности: [[Целитель]] и [[Ангел хранитель]]. 

Примечание 2: способность [[Менталист]] позволяет наносить урон по [[Рассудок|Рассудку]] цели, а не по [[Здоровье|Здоровью]].

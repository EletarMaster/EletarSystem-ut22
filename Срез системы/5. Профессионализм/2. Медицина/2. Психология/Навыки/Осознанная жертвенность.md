([[Типы способностей#Пассивная способность|П]]) [[Действия#Пассивный эффект|Пассивный эффект]]. 

1. Максимальный запас [[Здоровье|здоровья]] и [[Рассудок|рассудка]] персонажа увеличивается на 30 пунктов и еще на 15%.
2. Каждый раз, когда персонаж использует восстановление ресурсов цели с помощью [[Первая помощь|Первой помощи]], он также может передавать ей дополнительные ресурсы из своего запаса.
3. Можно передавать только значения кратные 5.
4. Переданное значение не может превышать количество восстановленного с помощью [[Первая помощь|Первой помощи]] ресурса.

([[Типы способностей#Пассивная способность|П]]) [[Действия#Пассивный эффект|Пассивный эффект]]. 

1. Когда персонаж пересобирает заново что-то с помощью проверки ремесла: [[Статейки/Броски/5. Профессионализм/Инженер|Инженер]], он улучшает [[Качество объектов|качество]] этого объекта на 1 ступень. 
2. Для успешной пересборки, ему сперва необходимо полностью разобрать объект, а затем собрать заново.
3. Улучшение качества влечет за собой улучшение параметров объекта. 
4. Каждый объект может быть пересобран таким образом лишь один раз.
5. Помимо этого, один раз за игровую сессию, персонаж может попытаться починить до [[Состояние объектов#Поношенное|состояния: поношенное]]: полностью сломанный объект с [[Состояние объектов#Убитое|состоянием: убитое]].

# Проверки

1. Ближний бой ([[Атака бб]]): +2.
2. Персонаж получает бонус +4 на все атаки [[Атака#Ближний Бой|Ближнего боя]] по целям, над которыми у него есть [[Превосходство]].
# Ресурсы

1. Нет бонусов.
# Пассивные параметры

2. Персонаж наносит на 10 пунктов Урона больше всеми атаками [[Атака#Ближний Бой|Ближнего боя]] целям, над которыми у него есть [[Превосходство]]. 
3. [[Критический Удар#Критический Урон|Критический урон]] всех атак [[Атака#Ближний Бой|Ближнего боя]], наносимых персонажем по целям, над которыми у него есть [[Превосходство]], увеличивается на 20. 
# Сопротивления

1. Нет бонусов.
# Изменение характеристик

1. Теперь персонаж может выбрать [[Интеллект]] или [[Обаяние]] в качестве характеристики для бросков Ближнего боя ([[Атака бб]]).
2. Теперь персонаж может выбрать [[Интеллект]] или [[Обаяние]] качестве характеристики для нанесения Урона в Ближнем бою.
3. Теперь персонаж может выбрать [[Интеллект]] или [[Обаяние]] качестве характеристики для использования: [[Защита#Ближний бой|Защиты - Ближний бой]].

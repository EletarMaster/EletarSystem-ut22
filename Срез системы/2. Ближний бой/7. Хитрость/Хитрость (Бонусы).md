# Проверки

1. Ближний бой ([[Атака бб]]): +2.
2. Персонаж получает бонус +8 на все броски Ближнего боя ([[Атака бб]]) по целям, над которыми у него есть [[Превосходство]].

3. Нет бонусов.
# Пассивные параметры

1. Персонаж наносит на 40 пунктов [[Урон|Урона]] больше всеми [[Атака#Ближний Бой|атаками ближнего боя]] целям, над которыми у него есть [[Превосходство]]. 
2. [[Критический Удар#Критический Урон|Критический урон]] всех [[Атака#Ближний Бой|атак ближнего боя]], наносимых персонажем по целям, над которыми у него есть [[Превосходство]], увеличивается на 40. 
# Сопротивления

1. Нет бонусов.
# Изменение характеристик

1. Теперь персонаж может выбрать [[Интеллект]] или [[Обаяние]] в качестве характеристики для бросков Ближнего боя ([[Атака бб]]).
2. Теперь персонаж может выбрать [[Интеллект]] или [[Обаяние]] качестве характеристики для нанесения Урона в Ближнем бою ([[Атака бб]]).
3. Теперь персонаж может выбрать [[Интеллект]] или [[Обаяние]] качестве характеристики для использования: [[Защита#Ближний бой|Защиты - Ближний бой]].

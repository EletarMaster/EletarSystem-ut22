# Видение смерти:

([[Типы способностей#Активная способность|А]]) [[Действия#Свободное действие|Свободное действие]]. [[Использование#1 Раз за Игровую Сессию|(1р/Сессия)]]. Стоимость: 0 Ресурсов.

В момент использования:
1. Персонаж совершает бросок любого [[Знания]].
2. В зависимости от результата, он может получить от Мастера честный ответ на вопрос "Что готовит смерть?".
3. Результат = 15: Персонаж получает видение о чей-то смерти.
4. Результат = 20: Персонаж получает видение о предположительной смерти того на кого смотрит персонаж.
5. Результат = 30: Персонаж получает видение о наиболее вероятной смерти того на кого смотрит персонаж.
6. Результат = 40+: Персонаж получает видение о наиболее вероятной смерти того на кого смотрит персонаж и получает от Мастера подсказку как этого избежать. Если это возможно...

Свойства эффекта:
1. [[Вид Воздействия#Нейтральный Эффект|Нейтральный]] - [[Тип Воздействия#Мистический|Мистический]] эффект: [[Вмешательство в  судьбу]].
2. Источник: [[Источник#Смерть|Смерть]].
3. Длительность эффекта: [[Длительность#Мгновенно|Мгновенно]].

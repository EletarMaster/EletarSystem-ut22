# Фортификация:

([[Типы способностей#Активная способность|А]]) [[Действия#Свободное действие|Свободное действие]]. [[Использование#1 Раз за Игровую Сессию|(1р/Сессия)]]. Стоимость: 0 Ресурсов.

Ограничения: эту способность можно использовать только когда персонаж занят возведением укреплений или постройкой какой-то конструкции. 

Пока эффект активен:
1. Итоговое время необходимое для постройки снижается на 25%. 
2. Во время работы над конструкцией, персонаж не получает пункты истощения из-за усталости во время нее. 
3. Итоговая прочность постройки увеличивается на 25%.

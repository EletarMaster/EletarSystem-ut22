# Анализ боевых параметров:

([[Типы способностей#Действия e6bb62 Реакция|Р]]) [[Действия#^e6bb62|Реакция]]. [[Использование#1 Раз за Игровую Сессию|(1р/Сессия)]]. Стоимость: 0 Ресурсов.

Условие срабатывания: против персонажа использовали уникальную способность - атаку.

Пока эффект активен:
1. Вне зависимости от того смог ли персонаж защититься от способности - атаки вызвавшей срабатывание реакции, все последующие попытки атаковать его этой способностью автоматически являются неудачными. Даже если их используют другие персонажи. 
2. Максимальное количество приемов, которое персонаж может запомнить таким образом = 2. 

Свойства эффекта:
1. [[Вид Воздействия#Положительный Эффект|Положительный]] - [[Тип Воздействия#Ментальный|Ментальный]] эффект: [[Усиление]].
2. Источник: [[Источник#Психика|Психика]].
3. Длительность эффекта: [[Длительность#Постоянная|Постоянная]].
4. Этот эффект невозможно рассеять или снять с персонажа никакими способами, кроме способов воздействующих на его память. 
5. Персонаж может потратить 2 часа времени во время отдыха и забыть один из запомненных ранее приемов чтобы освободить место для новых.

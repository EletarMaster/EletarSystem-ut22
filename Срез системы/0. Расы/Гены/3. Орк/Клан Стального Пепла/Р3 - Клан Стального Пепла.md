# Подрыв из ручной баллисты:

([[Типы способностей#Действия e6bb62 Реакция|Р]]) [[Действия#^e6bb62|Реакция]]. [[Использование#1 Раз за Игровую Сессию|(1р/Сессия)]]. Стоимость: 0 Ресурсов.

Условие срабатывания: персонаж наносит кому-то [[Урон]] в ближнем бою. 

В момент использования:
1. Сразу же после нанесения [[Урон|Урона]] по цели в ближнем бою, персонаж взрывает ручную баллисту. 
2. Атака наносит дополнительные 50 пунктов [[Урон|Урона]] от источника: [[Источник#Огонь|Огонь]].
3. Кроме того ВЕСЬ [[Урон]] от этой атаки становится: [[Урон#Тяжелое Оружие|Тяжелым оружием]]. 

Свойства эффекта:
1. [[Вид Воздействия#Негативный Эффект|Негативный]] - [[Тип Воздействия#Физический|Физический]] эффект: [[Нанесение урона]].
2. Источник: [[Источник#Огонь|Огонь]].
3. Длительность эффекта: [[Длительность#Мгновенно|Мгновенно]].

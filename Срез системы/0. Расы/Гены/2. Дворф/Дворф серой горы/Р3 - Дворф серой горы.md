# Оплот серой горы:

([[Типы способностей#Активная способность|А]]) [[Действия#Свободное действие|Свободное действие]]. [[Использование#1 Раз за Игровую Сессию|(1р/Сессия)]]. Стоимость: 0 Ресурсов.

Пока эффект активен:
1. Персонаж получает [[Снижение урона]]: 10 пунктов (Любой). 
2. Персонаж получает иммунитет к временным [[Вид Воздействия#Негативный Эффект|Негативным]] - [[Тип Воздействия#Мистический|Мистическим]] эффектам.
3. Их длительность не должна превышать: [[Длительность#До конца сцены|до конца сцены]] или 10 минут времени вне сцены. 
4. Если он был подвержен им до применения способности их эффект тут же рассеивается. 

Свойства эффекта:
1. [[Вид Воздействия#Положительный Эффект|Положительный]] - [[Тип Воздействия#Мистический|Мистический]] эффект: [[Усиление]].
2. Источник: [[Источник#Земля|Земля]].
3. Длительность эффекта: [[Длительность#2 Раунда|2 Раунда]].

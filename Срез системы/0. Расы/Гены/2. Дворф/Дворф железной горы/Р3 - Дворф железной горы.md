# Остатки великого архива:

([[Типы способностей#Активная способность|А]]) [[Действия#Свободное действие|Свободное действие]]. [[Использование#1 Раз за Игровую Сессию|(1р/Сессия)]]. Стоимость: 0 Ресурсов.

В момент использования:
1. Персонаж совершает бросок любого [[Знания]].
2. В зависимости от результата, он может получить от Мастера честный ответ на 1 ЛЮБОЙ вопрос, даже тот на который персонаж физически не мог знать ответ. 
3. Результат = 15: Краткий ответ.
4. Результат = 20: Развернутый ответ.
5. Результат = 30: Полный ответ.
6. Результат = 40+: Полный ответ и игрок получает информацию о других крупицах великого архива, если они есть в радиусе 100 км от него. 

Свойства эффекта:
1. [[Вид Воздействия#Нейтральный Эффект|Нейтральный]] - [[Тип Воздействия#Ментальный|Ментальный]] эффект: [[Усиление]].
2. Источник: [[Источник#Психика|Психика]].
3. Длительность эффекта: [[Длительность#Мгновенно|Мгновенно]].

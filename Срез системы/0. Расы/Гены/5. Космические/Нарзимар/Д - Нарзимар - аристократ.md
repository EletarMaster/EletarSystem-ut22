# Крылья Нарзимара:

1. Персонаж обладает огромными кожаными крыльями.
2. Он может передвигаться используя полет на свою скорость. 
# Клыки Нарзимара - аристократ:

1. Урон: 4d10 + Тел / Лов. Крит: 18-20 (+30). Пробивание брони: 20. Дальность: Ближняя. Тип: Клыки; Естественное; Одноручное. Оружие ближнего боя. Вес: 0. Цена: 0 мм.
# Когти Нарзимара - аристократ:

1. Урон: 2d8 + Тел / Лов. Крит: 18-20 (+20). Дальность: Ближняя. Тип: Когти; Естественное; Одноручное. Оружие ближнего боя. Вес: 0. Цена: 0 мм.
2. Атакуя этими когтями, персонаж может использовать базовый прием: [[Атака не основной рукой]] без штрафов.

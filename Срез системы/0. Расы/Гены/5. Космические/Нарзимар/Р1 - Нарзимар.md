# Клятва нарзимарского рыцаря

1. В любой момент повествования, персонаж может принести клятву. 
2. Если он нарушит ее то общество узнает, не спрашивайте как, узнает обязательно. 
3. В моменты, когда персонажу необходимо напрямую придерживаться этой клятвы, он получает [[Превосходство]] на все броски сопротивления [[Вид Воздействия#Негативный Эффект|Негативным]] [[Тип Воздействия#Ментальный|Ментальным]] эффектам. 
4. Персонаж может одновременно дать только 3 клятвы. 
5. Он не может своевольно отменить клятву - это может сделать только один из магистров нарзимарского ордена рыцарей. 

Примечание: превосходство работает не на абстрактные обстоятельства, а на конкретные угрозы и испытания мешающие персонажу исполнить клятву.

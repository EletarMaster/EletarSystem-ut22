# Когти Артиклоида:

1. Урон: 2d10 + Тел / Лов. Крит: 18-20 (+20). Дальность: Ближняя. Тип: Когти; Естественное; Одноручное. Оружие ближнего боя. Вес: 0. Цена: 0 мм.
2. Атакуя этими когтями, персонаж может использовать базовый прием: [[Атака не основной рукой]] без штрафов. 
# Жажда поглощения:

1. [[Использование#1 Раз за Игровую Сессию|Один раз за игровую сессию]] персонаж может сожрать полностью беззащитного противника потратив на это [[Действия#Стандартное действие|Стандартное действие]].
2. Если он это делает, он может совершить бросок: [[Пробуждение|Пробуждения]].
3. Персонаж получает усиления в зависимости от результата.
4. Результат = 15: В конце сессии персонаж получает 5% к прогрессу автоматической прокачки характеристик: [[Тело]] и [[Характер]].
5. Результат = 20: В конце сессии персонаж получает 10% к прогрессу автоматической прокачки характеристик: [[Тело]] и [[Характер]].
6. Результат = 30: В конце сессии персонаж получает 20% к прогрессу автоматической прокачки характеристик: [[Тело]] и [[Характер]].
7. Результат = 40+: В конце сессии персонаж получает 30% к прогрессу автоматической прокачки характеристик: [[Тело]] и [[Характер]].
8. Кроме того, иногда при поглощении уникальных противников, Мастер может награждать персонажа способностями.

2. [[Обаяние]]: +2.

# Когти Артиклоида:

1. Урон: 2d10 + Тел / Лов. Крит: 18-20 (+20). Дальность: Ближняя. Тип: Когти; Естественное; Одноручное. Оружие ближнего боя. Вес: 0. Цена: 0 мм.
2. Атакуя этими когтями, персонаж может использовать базовый прием: [[Атака не основной рукой]] без штрафов. 
# Жажда поглощения:

1. [[Использование#1 Раз за Игровую Сессию|Один раз за игровую сессию]] проводя с кем-то время во врем занятия интимом, персонаж может попытаться подселить в партнера новую жизнь - часть себя. 
2. Если он это делает, он должен совершить [[Встречная проверка|встречный]] бросок [[Пробуждение|Пробуждения]] против [[Тело|Тела]] цели. 
3. Персонаж получает усиление за КАЖДУЮ цель провалившую бросок сопротивления во время участия в процессе интима. 
4. Результат = 15: В конце сессии персонаж получает +2 к прогрессу автоматической прокачки характеристик: [[Тело]] и [[Обаяние]].
5. Результат = 20: В конце сессии персонаж получает +4 к прогрессу автоматической прокачки характеристик: [[Тело]] и [[Обаяние]].
6. Результат = 30: В конце сессии персонаж получает +6 к прогрессу автоматической прокачки характеристик: [[Тело]] и [[Обаяние]].
7. Результат = 40+: В конце сессии персонаж получает +8 к прогрессу автоматической прокачки характеристик: [[Тело]] и [[Обаяние]].
8. Кроме того, иногда при участии в каких-то особых мероприятиях такой тематики, Мастер может награждать персонажа способностями.

# Каста: Отступник Крит

([[Типы способностей#Пассивная способность|П]]) [[Действия#Пассивный эффект|Пассивный эффект]] 

1. [[Тело]]: +1.
2. [[Ловкость]]: +1.
3. [[Интеллект]]: +1.
4. [[Характер]]: +1.
5. [[Обаяние]]: +1.
6. [[Восприятие]]: +1. 
7. Получаемый персонажем [[Опыт]] увеличивается на 5%. 


Примечание: выбирать этот подвид стоит на свой страх и риск, так как в обществе Критианцев такие персонажи являются крайне чужеродными элементами. Они не следуют их традициям и порицаются обществом. Мастер может запретить персонажу выбирать этот подвид Критианца. Также этот подвид может раскрыться в ходе сюжета персонажа.

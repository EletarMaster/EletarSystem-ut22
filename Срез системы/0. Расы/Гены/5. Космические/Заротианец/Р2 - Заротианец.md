# ГЕН: Естественные инструменты

([[Типы способностей#Пассивная способность|П]]) [[Действия#Пассивный эффект|Пассивный эффект]] 

1. [[Тело]]: +1.
2. [[Ловкость]]: +1.
3. [[Восприятие]]: +1. 
4. Персонаж может гораздо эффективнее пользоваться своими клыками, когтями и хвостом. 

# Заротианские клыки:

1. Урон: 3d10 + Тел / Лов. Крит: 18-20 (+30). Пробивание брони: 20. Дальность: Ближняя. Тип: Клыки; Естественное; Одноручное. Оружие ближнего боя. Вес: 0. Цена: 0 мм.

# Заротианские когти:

1. Урон: 2d8 + Тел / Лов. Крит: 18-20 (+20). Дальность: Ближняя. Тип: Когти; Естественное; Одноручное. Оружие ближнего боя. Вес: 0. Цена: 0 мм.
2. Атакуя этими когтями, персонаж может использовать базовый прием: [[Атака не основной рукой]] без штрафов. 

# Заротианский хвост:

1. Урон: 2d10 + Тел / Лов. Крит: 20 (+30). Дальность: 2 клетки. Тип: Хвост; Естественное; Одноручное. Оружие ближнего боя. Вес: 0. Цена: 0 мм.
2. Персонаж может использовать свой хвост в качестве полноценной конечности.

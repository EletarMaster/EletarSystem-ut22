# Подвид: Дракон

([[Типы способностей#Пассивная способность|П]]) [[Действия#Пассивный эффект|Пассивный эффект]] 

1. [[Тело]]: +4.
2. [[Ловкость]]: +4.
3. [[Интеллект]]: +4.
4. [[Обаяние]]: +4.
5. Иммунитет к урону и негативным эффектам источника: [[Источник#Огонь|Огонь]]. 

# Крылья Дракона:

1. Персонаж обладает драконьими крыльями.
2. Он может передвигаться используя полет на свою скорость +4 [[Клетка|клетки]]. 
3. [[Использование#1 Раз за Сцену|Один раз за сцену]] персонаж может взлететь не провоцируя свободные атаки и атаки - реакции. 

Примечание: выбирать этот подвид стоит на свой страх и риск, так как за персонажем 100% будет вестись охота. Мастер может запретить персонажу выбирать этот подвид Заротианца. Также этот подвид может раскрыться в ходе сюжета персонажа.

# Искра титанов: Покров Нуминэкса:

([[Типы способностей#Активная способность|А]]) [[Действия#^d81ed1|Малое действие]] [[Использование#1 Раз за Сцену|(1р/Сцена)]]. Стоимость: 30 ([[Выносливость|Вын]]).

Пока эффект активен:
1. Сразу же после использования, персонаж попадает под эффект: [[Незаметность]], даже если рядом нет [[Укрытие|укрытия]]. 
2. Если кто-то видел как персонаж делает это, он получает возможность совершить автоматический [[Встречная проверка|встречный]] бросок [[Внимательность|Внимательности]] против результата броска [[Плутовство|Плутовства]] персонажа.  
3. Используя эту способность, цель получает ТОЛЬКО Усиления от незаметности, без ослаблений. 

Усиления незаметности:
1. Цель не видно, он не может быть выбран в качестве цели для атаки или иного воздействия. 
2. Она также получает [[Превосходство]] на любые воздействия, где ее состояние могло бы позволить ей действовать эффективнее. 
3. Результат броска [[Плутовство|Плутовства]] персонажа является сложностью броска [[Внимательность|Внимательности]] для окружающих чтобы увидеть персонажа.
4. Его агрессивные действия и атаки не рассеивают с него эффект: [[Незаметность]], но каждый раз, когда персонаж их совершает в присутствии кого-то, находясь в этом состоянии, окружающие получают бесплатный бросок: [[Внимательность|Внимательности]] в качестве попытки обнаружить его. 

Свойства эффекта:
1. [[Вид Воздействия#Положительный Эффект|Положительный]] - [[Тип Воздействия#Ментальный|Мистический]] эффект: [[Изменение]].
2. Источник: [[Источник#Тьма|Тьма]].
3. Длительность эффекта: [[Длительность#2 Раунда|2 Раунда]].

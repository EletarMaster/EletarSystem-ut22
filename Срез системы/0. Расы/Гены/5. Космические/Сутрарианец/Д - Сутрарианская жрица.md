# Подвид: Сутрарианская жрица

([[Типы способностей#Пассивная способность|П]]) [[Действия#Пассивный эффект|Пассивный эффект]] 

1. [[Интеллект]]: +2.
2. [[Характер]]: +2. 
3. [[Обаяние]]: +2. 
4. [[Восприятие]]: +2. 
5. [[Энергия]]: +50 пунктов.
6. [[Энергия]]: +30%. 

# Хватка сутры:

1. Персонаж обладает двумя дополнительными руками.
2. Он может полноценно пользоваться ими так, будто все эти руки основные и никогда не получает штрафов за использование не основной руки. 
# Всевидящее око сутры:

1. Третий глаз, наделяющий сутрарианок могуществом и магическими способностями. 
2. [[Использование#1 Раз за Сцену|Один раз за сцену]] персонаж может совершить бросок любого [[Знания]].
3. В зависимости от результата, он может получить от Мастера честный ответ на 1 ЛЮБОЙ вопрос, даже тот на который персонаж физически не мог знать ответ. 
4. Результат = 15: Краткий ответ.
5. Результат = 20: Развернутый ответ.
6. Результат = 30: Полный ответ.
7. Результат = 40+: Полный ответ и игрок перезаряжает использование этой способности. 

Свойства эффекта:
1. [[Вид Воздействия#Нейтральный Эффект|Нейтральный]] - [[Тип Воздействия#Ментальный|Ментальный]] эффект: [[Вмешательство в  судьбу]].
2. Источник: [[Источник#Психика|Психика]].
3. Длительность эффекта: [[Длительность#Мгновенно|Мгновенно]].

Примечание: этот подвид сутрарианца может быть выбран только для персонажа женского пола.

# Золотой лик:

([[Типы способностей#Активная способность|А]]) [[Действия#Свободное действие|Свободное действие]]. [[Использование#1 Раз за Игровую Сессию|(1р/Сессия)]]. Стоимость: 0 Ресурсов.

В момент использования:
1. Персонаж совершает [[Встречная проверка|встречный]] бросок [[Интриганство|Интриганства]]; [[Манипуляция|Манипуляции]]; [[Присутствие|Присутствия]]; [[Компромисс|Компромисса]]; [[Привлекательность|Привлекательности]] против [[Характер|Характера]] целей. 
2. Применение этой способности [[Агрессивное действие#Не считается агрессивным действием|не считается агрессивным действием]]. 
# В случае успешного применения

Пока эффект активен:
1. Сразу же после использования, цель попадает под эффект: [[Очарование]]. 
2. Цель перестает быть враждебной по отношению к персонажу (если была до этого).
3. Помимо этого, все социальные проверки получают против нее бонус +4. 

Свойства эффекта:
1. [[Вид Воздействия#Негативный Эффект|Негативный]] - [[Тип Воздействия#Мистический|Мистический]] эффект: [[Контроль разума]].
2. Источник: [[Источник#Свет|Свет]].
3. Длительность эффекта: [[Длительность#2 Раунда|2 Раунда]]

Параметры применения:
1. Дальность: 10 клеток вокруг персонажа. 
2. Шаблон применения: 10 клеток вокруг персонажа. 
3. Количество целей: Все цели по выбору персонажа в шаблоне.
# Досрочное избавление от эффекта

1. Цель получает автоматический бросок [[Характер|Характера]] - [[Сложность|СЛ]]: 30 в начале каждого своего [[Ход|хода]], пока находится под действием эффекта. 
2. Любой полученный [[Урон#Внешний|внешний урон]] тут же рассеивает этот эффект.

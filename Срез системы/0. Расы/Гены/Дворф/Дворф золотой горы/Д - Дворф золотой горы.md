# Золотой банк

1. Персонаж может заверить любую заключенную сделку "Золотым банком"
2. В случае, если кто-то из участников сделки включая персонажа нарушит ее условия, он будет вынужден нести ответственность перед золотым банком. 
3. Не спрашивайте как это работает "...лик солнца все видит..."

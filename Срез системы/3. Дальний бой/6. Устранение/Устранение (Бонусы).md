# Проверки

1. Дальний бой ([[Атака дб]]): +2.
2. Персонаж получает бонус +4 на все атаки [[Атака#Дальний Бой|Дальнего боя]] по целям, над которыми у него есть [[Превосходство]].
# Ресурсы

1. Нет бонусов.

# Пассивные параметры

1. Персонаж наносит на 20 пунктов [[Урон|Урона]] больше всеми атаками [[Атака#Дальний Бой|Дальнего боя]] целям, над которыми у него есть [[Превосходство]]. 
2. [[Критический Удар#Критический Урон|Критический Урон]] атак [[Атака#Дальний Бой|Дальнего боя]] персонажа: +20.
# Сопротивления

1. Нет бонусов.

# Изменение характеристик

1. Теперь персонаж может выбрать [[Интеллект]] или [[Обаяние]] в качестве характеристики для бросков Дальнего боя ([[Атака дб]]).
2. Теперь персонаж может выбрать [[Интеллект]] или [[Обаяние]] в качестве характеристики для нанесения Урона в Дальнем бою.

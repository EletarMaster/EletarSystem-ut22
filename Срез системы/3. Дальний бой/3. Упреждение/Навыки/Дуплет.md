Условие срабатывания: персонаж атакует противника в [[Атака#Дальний Бой|Дальнем бою]], используя [[Действия#Стандартное действие|Стандартное]] или [[Действия#Малое действие|Малое]] действие. 

В момент использования:
1. Персонаж тут же повторяет ту же самую атаку со всеми ее усилениями и параметрами. 
2. Это работает даже если у атаки есть ограничение по количеству раз использования.
3. Персонаж тратит только ресурсы, указанные в этой способности, он не тратит ресурсы за саму атаку и ее усиления повторно.
4. Персонаж может выбрать другую цель для этой атаки.
5. Персонаж не тратит боеприпасы на второй выстрел.

Свойства эффекта:
1. [[Вид Воздействия#Негативный Эффект|Негативный]] - [[Тип Воздействия#Физический|Физический]] эффект: [[Усиление]].
2. Источник: [[Источник#Физический|Физический]].
3. Длительность эффекта: [[Длительность#Мгновенно|Мгновенно]].

Примечание: сама по себе эта способность НЕ ЯВЛЯЕТСЯ атакой дальнего боя - это усиление, которое позволяет ПОВТОРИТЬ атаку дальнего боя.
